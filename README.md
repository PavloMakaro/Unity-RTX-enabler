# Ray Tracing Enabler for Unity HDRP

One-click plugin to enable **Ray Tracing** in **HDRP** projects.

**Unity 2023.2+ / Unity 6+** · **DX12** · **RTX / RX 6000+**

---

## Features
- One-click enable (Medium preset)
- UI: Quality (Low/Med/High), toggles, intensity sliders
- Auto: DX12, disable static batching, Global Volume + RT effects

---

## Install (1 file)

```bash
Assets/Editor/RayTracingEnabler/RayTracingEnabler.cs
Paste full code from below → Reload Unity.
Use
